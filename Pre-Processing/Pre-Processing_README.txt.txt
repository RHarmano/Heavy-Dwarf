This folder contains the pre-processing scripts used to process the .csv files containing the dSph galaxy data
It converts the .csv files to a list containing tuples of the processed data cubes and the corresponding labels
and saves them to a numpy file type
To operate this code specify the file path to the directory containing the .csv files and the folder where the
.npy files will be saved.
Run this code either by running the script in an IDE or in the command window
